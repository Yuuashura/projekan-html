## API Reference

#### Get all items



```http
  Base API : https://yuuashura-api.vercel.app/
```

```http
  GET /{query}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `string` | `string` | **Nanti Ada Endpointnya di API** |

#### Get item

```http
  GET /{slug}/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
